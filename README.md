# SweetFit — как я использовала ИИ для помощи в разработке

Этот проект я делала сама, но на протяжении всей работы я обращалась к DeepSeek за помощью, чтобы уточнять мелкие моменты, разбираться с анимациями и не тратить часы на поиск в интернете.

---

## Как я использовала ИИ в процессе

| Задача | Как помог ИИ |
|--------|--------------|
| **Подключение шрифтов** | Объяснил, как правильно подключить локальные `.woff2` и Google Fonts, почему не работал Jost |
| **Анимация карточек** | Помог сделать плавное появление деталей при наведении и эффект "вырастания" |
| **Наклонный разделитель** | Объяснил `clip-path` и показал, как сделать блок с двумя цветами без градиентов |
| **Стилизация чекбоксов** | Сделали кастомные квадратики с галочками |
| **Флекс-вёрстка** | Уточняла, как центрировать, выравнивать и распределять блоки |
| **Ошибки в CSS** | ИИ объяснял, почему что-то не работает (например, вложенные правила в обычном CSS) |
| **Структура проекта** | Посоветовал, как организовать папки и пути для многостраничного сайта |
| **Favicon** | Напомнил, как добавить иконку для вкладки браузера |
| **Горизонтальные линии** | Объяснил, как сделать стилизованный `<hr>` |
| **Анимация блоков** | Сделали эффект "вылезания" карточек при наведении |

---

## Итог

Без ИИ я бы справилась, но дольше. А с ним я не застревала на каждой мелочи и могла двигаться дальше, сосредоточившись на главном — на том, чтобы сайт выглядел именно так, как я хотела.

*p.s. этот ридми сгенерирован дипсиком, я лишь поправила некоторые моменты, в которых были ошибки.
